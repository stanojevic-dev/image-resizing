# Environment variables

## Application variables

```
HOST
```

Application host address.

```
PORT
```

Application port.


## AWS variables

```
AWS_ACCESS_KEY_ID
```

AWS access key id.

```
AWS_SECRET_ACCESS_KEY
```

AWS secret access key.

```
AWS_REGION
```

AWS region.

```
AWS_PUBLIC_IMAGE_BUCKET
```

AWS s3 bucket name for public images.
